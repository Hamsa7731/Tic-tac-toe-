🎮 Tic-Tac-Toe Game (Java Console)

📌 Description

This is a simple Tic-Tac-Toe game built using Java.
It is a console-based application where two players can play the game by entering positions from 1 to 9.

---

🚀 Features

- Two-player game (Player X and Player O)
- Input positions from 1–9
- Automatically checks:
  - Winner 🏆
  - Draw 🤝
- Simple and easy-to-understand code

---

🛠️ Technologies Used

- Java
- Scanner class (for user input)
- Arrays and loops

---

▶️ How to Run

1. Open your IDE (Eclipse / IntelliJ / VS Code)
2. Create a new Java project
3. Create a class named:
   TicTacToe.java
4. Copy and paste the code
5. Run the program ▶️

---

🎮 How to Play

- The board positions are like this:

1 | 2 | 3
--+---+--
4 | 5 | 6
--+---+--
7 | 8 | 9

- Player X starts first
- Enter a number (1–9) to place your mark
- Players take turns
- First to align 3 marks wins 🎉
